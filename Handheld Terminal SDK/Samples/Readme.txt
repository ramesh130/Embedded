
The samples folder contains the programs to illustrate the usage of the APIs of different peripherals.
The following are the contents of this folder-

1. Buzzer
   Include files common to the different variants of the board.
   
2. GPIO
   Source files common to the different variants of the board.
   
3. GSM

4. I2C_EEPROM

5. KeyPad

6. LCD

7. PC_Comm

8. Printer

9. RTC

10. SmartCard

11. SPI_Flash

12. Template
    A project template to create a new project.

13. Timer

14. Utils
   
15. Readme.txt
    This document.
